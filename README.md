# Employee Management System – SQL Project

## 📌 Project Overview

The **Employee Management System (EMS)** is a SQL-based database project designed to efficiently manage employee-related operations within an organization.
The system centralizes employee records, department details, payroll processing, salary structures, qualifications, and leave management using relational database concepts.

This project demonstrates:

* Relational Database Design
* SQL Query Writing
* Data Integrity using Constraints
* Analytical Reporting using SQL

---

# 🎯 Business Problem

Many organizations struggle with:

* Manual employee record management
* Payroll calculation errors
* Duplicate or inconsistent data
* Difficulty tracking employee qualifications and leaves
* Lack of centralized HR management

This project solves these issues by developing a centralized Employee Management System using SQL.

---

# 🎯 Project Objectives

* Develop a centralized Employee Management System
* Manage employee, payroll, qualification, and leave data efficiently
* Maintain data consistency using relational database concepts

# 🗂️ Database Tables

The project contains **6 tables**:

| Table Name    | Description                            |
| ------------- | -------------------------------------- |
| JobDepartment | Stores department and job role details |
| Employee      | Stores employee information            |
| SalaryBonus   | Stores salary and bonus information    |
| Qualification | Stores employee qualifications         |
| Leaves        | Stores leave records                   |
| Payroll       | Stores payroll transactions            |
